Proyecto backend con node.js 

PPeliculas:
plataforma o foro con informacion del mundo de las peliculas, contiene peliculas, actores, generos y toda su informacion, adicional con un filtro de busqueda para encontrar la informacion de manera adecuada

Herramienta:
Node.js
MVCS (modelo-vista-controlador-servicios)
MYSQL

Elementos:
•	Servidores: maneja solicitudes de los usuarios
•	Bases de datos: almacena y organiza los datos
•	APIs: permiten la comunicación entre diferentes sistemas y aplicaciones
•	Lenguaje de programación: la base para desarrollar la lógica del backend
